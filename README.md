## ✨ 代码结构

项目使用Flask blueprints, app factory pattern, 文件代码结构如下：

```bash
< PROJECT ROOT >
   |
   |-- src/
   |    |
   |    |-- Add                           # 增加字典词汇类
   |    |-- App                 # 应用运行的main方法主类
   |    |-- DataWindow                 # 数据窗体展现类
   |    |-- Delect                 # 删除字典词汇类
   |    |-- Modify                 # 修改字典词汇类

   |-- ************************************************************************
```
